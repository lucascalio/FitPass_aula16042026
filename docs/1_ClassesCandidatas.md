# Requisitos Funcionais do Sistema

## RF01 — Cadastro de Alunos
O sistema deve permitir cadastrar novos alunos contendo:
- dados pessoais  
- contato  
- endereço  
- plano contratado  

## RF02 — Gerenciamento de Planos
O sistema deve permitir:
- criar planos  
- editar planos  
- ativar planos  
- desativar planos  

Exemplos:
- mensal  
- trimestral  
- anual  
- musculação  
- funcional  

## RF03 — Controle de Pagamentos
O sistema deve:
- registrar pagamentos realizados na recepção:
  - dinheiro  
  - cartão  
  - PIX  
- gerar boletos/cartões para pagamentos online  

## RF04 — Regularidade do Aluno
O sistema deve verificar automaticamente se o aluno está com a mensalidade em dia.

## RF05 — Controle de Acesso
O sistema deve:
- integrar-se à catraca  
- validar a entrada do aluno por RFID  

## RF06 — Agendamento de Aulas
O aluno deve poder:
- visualizar horários  
- reservar vagas nas aulas disponíveis  

## RF07 — Lista de Presença
Instrutores devem:
- registrar presença dos alunos nas aulas  

## RF08 — Avaliação Física
Instrutores devem:
- registrar avaliações físicas:
  - peso  
  - IMC  
  - percentual de gordura  
- anexar arquivos, se necessário  

## RF09 — Relatórios Gerenciais
O gerente deve poder emitir relatórios como:
- inadimplência  
- alunos ativos  
- histórico de acessos  
- ocupação das aulas  

## RF10 — Notificações
O sistema deve enviar notificações ao aluno sobre:
- vencimento de mensalidade  
- confirmação de agendamento  
- liberação de nova avaliação física
